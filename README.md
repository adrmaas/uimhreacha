# uimhreacha
uimhreacha is a vibe coding test project to deveolop an iOS app for tracking miscellaneous things.

## Uses

The app can be used to track miscellaneous activities like drinking a glass of water.  All tasks are currently user defined.

The app will ask about how you feel a three times a day: morning, afternoon, and evenning and send you a notification to prompt your mood.

The app can compare historical data that you track, your mood responses as well as Apple Health data like sleep and heart rate to look for correlations.

## Data

All data is private and stored in iCloud using Core Data for sharing across devices and maximum privacy.  Data is never available for third parties.

## Project
The project name is `uimhreacha` from the Irish for numbers.  The project files can be found in the `uimhreacha/` folder in this prepository.  The xcode project file can be found at `uimhreacha/uimhreacha.xcodeproj`.
